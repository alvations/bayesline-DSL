=== VarDial 2017 - Test Set Release ===

This archive contains the test sets for the ADI, DSL and GDI tasks at VarDial 2017. The CLP task evaluation is run through that task's repository.

In this folder you will find the following files:


** ADI/ADI-test-words.txt: Test set for the Arabic Dialect Identification (ADI) task. Each line is a sentence generated using Automatic Speech Recognition. This can be classified using the Arabic transcription training data ($lang.words).

** ADI/ADI-test-ivec.txt: Test set for the Arabic Dialect Identification (ADI) task. Each line is a 400 dimension ivector feature vector. This can be classified using the Arabic ivector training data ($lang.ivec).

** ADI/README.txt: Additional ADI info and link to the original audio files.

** DSL/DSL-test.txt: Test set for the Discriminating between Similar Languages (DSL) task. Each line consists of excerpts of journalistic texts written in the same languages 14 languages available in the training set.

** GDI/GDI-test.txt: Test set for the German Dialect Identification (GDI) task. Each line is an utterance extracted from transcripts of 4 German dialects.

** SUBMISSION.txt - Instructions for formatting your submissions.

** README.txt - This file.


All information regarding submitting your results and key dates are provided in SUBMISSION.txt, in this archive.
